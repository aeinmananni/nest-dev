
   تماما به فرم ماژولار خواهد بود nest ساختار 
   یعنی برای مثال : 
                   ما یک ماژول اپلیکشن خواهیم داشت که تما فیچر هایی را که داریم
                   زیر مجموعه ای از این قرار بگیرند

                  Application
          
     userModule     OrdersModule   chatModule          

-------------------------------------

با این دستور میتوانیم یک ماژول جدید اضافه کنیم
  اضافه میکند app.module.ts  و ان را درون  فایل app  با این دستور توی پوشه 

  nest g module  users

  nest g controller  users: برای اجرا کردن کنترلر

  nest g service  users : برای ایجاد کردن سروریس

  -------------------------------------------------
  برای اینکه بخواهیم از سرویس ماژول های دیگر در یک ماژول استفاده کنیم
  باید ماژولی که میخواهیم از سرویس ان استفاده کنیم را 
 کنیم imports درون فایل ماژول مبدا 

 ---------------------------------------------
   میتوانیم به دو صورت ماژول های خودرا تعریف کنیم nest  مادرون : 
                
                1- حالت استاتیک

                مثال : (
                            import { Module } from '@nestjs/common';
                            import { AppController } from './app.controller';
                            import { AppService } from './app.service';
                            import { UsersModule } from './users/users.module';

                            @Module({
                              imports: [UsersModule],
                              controllers: [AppController],
                              providers: [AppService],
                            })
                            export class AppModule {}

                        )


                 2- حالت داینامیک :  
                                    حالا اگر ما به صورت داینامیک ماژول خود را ایجاد میکردیم
                                    ان را وارد میکرد app.module.ts  درفایل  imports در قسمت 
                                    حالا اگر بخواهیم در هنگام اینپورت کردن یکسری تنظیمات اضافه
                                    کنیم باید این کاررا انجام دهیم
                                    ما ممکن است برای تنظیم تایپ دیتابیس از ان استفاده کنیم
                                    درواقع ما یک کانکشن ایجاد میکنیم تا تمام سرویس ها بتوانند از ان استفاده 
                                    کنند
